# Instruction

## Simple usage:
- There is a tool that helps you manage the server or the computer you host from.
- To start the tool press Win on your keyboard and type in `COL-Manager`

### Server manageing commands:

- To set the admin panel password you need to set it with this command: `col ad -p "Any passworld"`

- To start the server use this command:`col ser start p:"5000" n:"Any title for the website"`

- You can stop the server with this command: `col ser stop`

- To restart the server use this command:`col ser restart`

- You can delet all the chats from you computer with this command:`col dat delet`

- To encrypt a chat data use this command:`col dat -m encrypt "chat name"`

- To encrypt all the chats data use this command: `col dat -m encrypt --all`
